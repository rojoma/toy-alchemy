# Skills ディレクトリ

エージェントのスキル（プロンプト）をMarkdownファイルとして管理します。

## 構造

```
skills/
├── tutor/           # Tutorエージェントのスキル
│   ├── general.md   # 汎用フクロウ先生（フォールバック）
│   └── edu6th_math.md  # 小6算数（はいち先生スタイル）
├── referee/         # Refereeエージェントのスキル
│   └── referee_base.md  # 基本評価スキル（6スキル構造化評価）
└── README.md        # このファイル
```

## 新しいスキルの追加方法

### 1. Markdownファイルを作成
`skills/tutor/` に新しいファイルを追加します。

命名規則: `edu{学年}_{教科}.md`
- 例: `edu5th_math.md`（小5算数）
- 例: `edu6th_science.md`（小6理科）
- 例: `edu7th_math.md`（中1数学）

### 2. agent_core.py のマッピングに追加
`TUTOR_SKILL_MAP` に新しいエントリを追加:

```python
TUTOR_SKILL_MAP = {
    ("小学6", "算数"): "edu6th_math.md",
    ("小学5", "算数"): "edu5th_math.md",  # ← 追加
}
```

### 3. スキルファイルの推奨構造

```markdown
# {キャラクター名} — スキル定義

## 1. ペルソナ定義
## 2. ティーチングスタイル
## 3. 対話ルール
## 4. 知識ベース（カリキュラム）
## 5. NGルール
## 6. メタ指示（LINE向け追加ルール）
```

## スキル選択の仕組み

1. 子供のメッセージから教科を自動推定（算数/理科/国語）
2. 子供のプロファイルから学年を取得
3. 学年×教科でスキルマップを検索
4. 該当スキルがなければ `general.md` にフォールバック
