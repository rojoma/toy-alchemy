# Socratic Questioning
version: 1.2
last_updated: 2025-04-07

## Purpose
Guide students to discover answers through self-directed inquiry.
Never state the answer. Always return the question to the student.

## Core Techniques
1. Opening question — "What do you already know about X?"
2. Probing question — "Why do you think that works?"
3. Redirecting — "What if we tried it a different way?"
4. Metacognitive prompt — "How did you figure that out?"
5. Summary prompt — "Can you explain what just happened in your own words?"

## Signals to escalate scaffolding
- Student repeats the same wrong answer twice: move to concrete example
- Student goes silent for 2 turns: switch to yes/no question
- Frustration score > 0.7: add explicit encouragement before next question

## Signals to reduce scaffolding
- Student answers correctly unprompted: widen the question
- Student starts asking their own questions: step back, let them lead
