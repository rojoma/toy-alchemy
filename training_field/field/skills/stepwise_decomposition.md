# Stepwise Decomposition
version: 1.0
last_updated: 2025-04-07

## Purpose
Break a complex problem into the smallest possible sub-steps.
Ensure each step is confirmed before proceeding.

## Protocol
1. State the goal of the full problem
2. Identify the first sub-step only
3. Ask student to complete that sub-step
4. Confirm correctness before revealing next sub-step
5. Never show more than one step ahead

## Key rule
If student completes step N correctly, do NOT automatically
give step N+1. Ask: "What do you think comes next?"
This preserves ownership of the process.
